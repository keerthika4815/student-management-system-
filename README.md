Student Grade Management System
------------------------------------------------------------------------------------------------------------------------------------------------------------
Abstract:

The Student Grade Management System is a Python-based console application designed to manage and evaluate student academic performance efficiently. 
The system allows users to enter student details, calculate total marks and averages, assign grades automatically, and display structured student reports. 
It includes input validation and error handling to ensure data accuracy. 
This project demonstrates the use of core Python concepts such as functions, lists, dictionaries, loops, and exception handling
